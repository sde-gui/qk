5.3 DocumentWindow object

DocumentWindow.editor()
returns Editor object.
DocumentWindow.active_view()
returns current active document view in this window.
DocumentWindow.set_active_view(view)
makes view active, i.e. switches to its tab.
DocumentWindow.active_document()
returns current active document in this window, that is the document whose view is the active one.
DocumentWindow.set_active_document(doc)
makes active a view of document doc. It picks arbitrary view of doc if there are more than one in this window.
DocumentWindow.views()
returns list of all document views in this window.
DocumentWindow.documents()
returns list of all documents in this window.
DocumentWindow.is_active()
returns whether this window is the active one.
DocumentWindow.set_active()
makes this window active.
